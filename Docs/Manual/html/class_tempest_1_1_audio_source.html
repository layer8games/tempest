Detailed Description

An AudioSource is the location from which an AudioClip will play. This holds a clip, a position, and velocity, and can be told to start, stop or pause its play. Currently the source only supports one clip, but later could support many clips in the form of buffers. This is a more advanced way to use OpenAL that is currently not supported.
