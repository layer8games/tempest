Detailed Description

The GameObject may be the backbone of Tempest. This is the basic object that will exist in the game. It is the root of all rendered and interactive objects in the games. It is not a manager. This is an abstract class, so it is meant to be full implemented as needed. This holds the generic ideas of what makes up the most basic type of Object in the game Level.
