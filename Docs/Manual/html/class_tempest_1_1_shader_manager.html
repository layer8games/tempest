Detailed Description

A global manager for Shaders that are compiled for use. The idea behind this manager is that you can use it to re-use shaders. These are passed as pointers, so they can be shared and reused. Before a shader can be used, it has to be made active. That can be done on the Shader itself, this is more of a bank for all the shaders that have been created thus far.
