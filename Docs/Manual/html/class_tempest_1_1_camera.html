Detailed Description

Default Camera for the Engine. This Camera is fully operational. It controls the offset of the view port from the origin. When the position of the Camera is set, every other GameObject that is rendered is moved the negative of the position. If this Camera doesn't do what you need it to do, it is meant to be extended, with functions that are virtual.
