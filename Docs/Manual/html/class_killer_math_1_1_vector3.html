Detailed Description

A custom Vector3 class writen for the Killer Engine.

The Vector3 is a 3 element 3D Vector. The Vector3 consists of an x, y, and z component represented by an array of F32 values. They are refered as x, y, and z through documentation. In code, you can refere to them by x, y and z, due to an enum defined in Atom.h.
Unlike the Vector4, there is no w, so there is no difference between a position and a direction, like in the Point vs Vector4 classes. This class exists for when something doesn't need that extra data that comes with the w element. This can save on memory and computation. The Matrix3 class uses it, for example.
