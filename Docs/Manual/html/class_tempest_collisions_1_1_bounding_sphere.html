Detailed Description

The BoundingSphere is a basic bounding volume that can be used for simple, course collision tests. It is represented by a center position, located in the model space of the object it is attached to, and a radius, that describes its extent in all directions. TestCollision functions can be added for each type of boudning volume it can intersect with.

◆ GetGrowth()

real BoundingSphere::GetGrowth ( BoundingSphere &  other) const

Returns how much a Sphere would grow if it were combined with another Sphere. This size is not specified in specific units, but rather more closely reflect the surface area of the new sphere, so should be used to compare relative sizes, not specific sizes, since this would be unpredictable and far too much work to be with it. This function should be as a heuristic for tests between spheres.
