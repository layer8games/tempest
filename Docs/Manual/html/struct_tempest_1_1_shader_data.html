Detailed Description

All data needed to be able to compile a shader. Filepath is the path to the code file on the hard drive, type is the type of compiler needed by OpenGL.

Parameters
filePathis the path to the specific shader on the hard drive. This will be the source code for one shader stage, ie a vertex shader source file.
typeis the type of shader this file path is, ie VERTEX for a vertext shader file.
